189 send_robust (bool): Defaults to True. See Django signal docs.

192 update, use the timestamp from the DB record. Defaults to 

193 current time in UTC. This argument is optional for backward

308def _reconnect_to_db_if_needed(): # pragma: no cover 

309 """ 

310 Reconnects the db connection if needed. 

311 

312 This is important because Django only does connection validity/age checks as part of 

313 its request/response cycle, which isn't in effect for the consume-loop. If we don't 

314 force these checks, a broken connection will remain broken indefinitely. For most 

315 consumers, this will cause event processing to fail. 

316 """ 

317 has_connection = bool(connection.connection) 

318 requires_reconnect = has_connection and not connection.is_usable() 

319 if requires_reconnect: 

320 connection.connect()
